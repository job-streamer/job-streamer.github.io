What’s Notificator?

Notificator is the server that receive order from control-bus and call some API. It made wrapper as Apatche Camel so that specification is based on Apatche Camel.

What’s Notificator can do?

Notificator meet the demands that “when a job finish,wanna send a mail”.
details are notificator config.
If you want to do more, please refer to Apatche Camel.
