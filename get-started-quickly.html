Quickly

Control bus

  1. Get datomic-free:

    % unzip datomic-free-0.9.5130.zip
% cd datomic-free-0.9.5130
% bin/transactor config/samples/free-transactor-template.properties

  2. Download the latest version control bus:

  3. Run the contorl bus:

    % bin/control_bus

default port 45102

Management console

  1. Download the latest version management console:

  2. Run the management console:

    % bin/console

  2. Run the agent:

  2. Run the notificator:
